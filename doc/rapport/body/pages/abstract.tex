\begin{abstract}
  \centering
  Le problème considéré est le \GatheringProblem~: soit $n$ robots placés dans
  le plan, tous avec des positions différentes, les faire se rassembler en un
  point en un nombre fini de cycles. Le point de rassemblement n'est pas fixé à
  l'avance. Cette tâche constitue l'une des primitives de base pour le contrôle
  et la coordination de robots mobiles autonomes.
